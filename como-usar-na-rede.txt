======================================================
  COMO USAR O JB IMPORTES EM UMA REDE LOCAL
======================================================

Se você deseja que vários usuários acessem a loja e o painel administrativo
ao mesmo tempo, e que as alterações feitas por um usuário sejam vistas por
todos os outros, siga estas instruções:

1. CONFIGURAÇÃO DO SERVIDOR:

   - Certifique-se de que o Node.js está instalado no computador que servirá como servidor.
     - Para verificar, abra o terminal e digite: `node --version`
     - Se não estiver instalado, baixe em: https://nodejs.org/

   - Abra o terminal no diretório do projeto e instale as dependências:
     ```
     npm install
     ```

   - Inicie o servidor:
     ```
     npm start
     ```
     ou execute o arquivo `iniciar-servidor.bat` com duplo clique.

   - O servidor deve exibir uma mensagem como:
     ```
     Servidor rodando em http://192.168.1.2:3000
     Acesse http://192.168.1.2:3000 para visualizar a loja
     ```

2. ACESSO DOS OUTROS COMPUTADORES:

   - Em cada computador que irá acessar a loja, abra o navegador
   - Digite o endereço do servidor seguido da porta:
     http://[IP_DO_SERVIDOR]:3000
     
     Por exemplo:
     http://192.168.1.100:3000

   - Para acessar o painel administrativo, use:
     http://[IP_DO_SERVIDOR]:3000/admin.html
     
     Exemplo:
     http://192.168.1.100:3000/admin.html

3. SOLUÇÃO DE PROBLEMAS COMUNS:

   a) Se os produtos não estão aparecendo para todos:
      - Certifique-se de que todos estão usando o endereço IP do servidor
      - Verifique se o servidor está rodando
      - Recarregue a página após adicionar ou modificar produtos (F5)

   b) Se aparecer "Esta página não pode ser exibida":
      - Verifique se o endereço IP está correto
      - Verifique se o servidor está rodando
      - Certifique-se de que o firewall do Windows permite o acesso à porta 3000

   c) Para liberar a porta 3000 no firewall do Windows:
      - Painel de Controle > Sistema e Segurança > Firewall do Windows
      - Configurações avançadas > Regras de Entrada > Nova Regra
      - Selecione "Porta" > TCP e digite 3000 > Permitir a conexão
      - Dê um nome como "Servidor JB IMPORTES"

4. IMPORTANTE:

   - O arquivo "produtos.json" é onde todos os produtos são armazenados
   - Qualquer alteração feita por um usuário afeta todos os outros
   - Para fazer backup dos dados, copie o arquivo "produtos.json"
   - Para encerrar o servidor, pressione CTRL+C no terminal 

# Como usar o sistema JB IMPORTES na rede local

## Configuração do servidor

1. Certifique-se de que o Node.js está instalado no computador que servirá como servidor.
   - Para verificar, abra o terminal e digite: `node --version`
   - Se não estiver instalado, baixe em: https://nodejs.org/

2. Abra o terminal no diretório do projeto e instale as dependências:
   ```
   npm install
   ```

3. Inicie o servidor:
   ```
   npm start
   ```
   ou execute o arquivo `iniciar-servidor.bat` com duplo clique.

4. O servidor deve exibir uma mensagem como:
   ```
   Servidor rodando em http://192.168.1.2:3000
   Acesse http://192.168.1.2:3000 para visualizar a loja
   ```

## Problemas comuns e soluções

### Os produtos não aparecem para outros usuários

Se você adicionar produtos e eles não aparecerem para outros usuários:

1. **Verifique se o servidor está rodando corretamente**
   - O terminal deve mostrar que o servidor está rodando
   - Não feche o terminal onde o servidor está executando

2. **Certifique-se que todos os usuários estão acessando pelo IP do servidor**
   - Ao invés de usar "localhost", todos devem acessar pelo IP, ex: http://192.168.1.2:3000

3. **Verifique o firewall do computador servidor**
   - O firewall pode estar bloqueando conexões na porta 3000
   - Adicione uma exceção no firewall para permitir conexões nessa porta

4. **Atualize a página regularmente**
   - Os usuários devem atualizar a página (F5) para ver os novos produtos
   - O sistema tenta atualizar automaticamente a cada 60 segundos

### Imagens não são exibidas corretamente

1. Certifique-se de usar links de imagens válidos
   - Links externos (https://...) devem estar acessíveis
   - Para imagens locais, coloque-as na pasta "img/" e use o caminho "img/nome-da-imagem.jpg"

2. O sistema tentará mostrar uma imagem de "não disponível" quando não conseguir carregar

## Acesso ao painel administrativo

1. Clique no botão "Admin" na parte inferior da página principal
2. Use a senha: 123456
3. No painel administrativo você pode:
   - Adicionar novos produtos
   - Editar produtos existentes
   - Remover produtos

## Importante

- O servidor precisa estar em execução para que o sistema funcione corretamente
- Quando o servidor é desligado, novos dados não são sincronizados
- Todos os dados são armazenados no arquivo `produtos.json` no servidor
